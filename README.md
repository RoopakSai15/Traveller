# 🌍 Travel Planner App

An interactive travel planning application that allows users to search routes, view weather forecasts, and save trips. Premium features are unlocked through Stripe payments.

## 💡 Use Cases

- Plan multi-city trips with map support
- View current & future weather using OpenWeatherMap API
- Upgrade to premium with Stripe checkout
- Authenticated user dashboard

## 🔌 APIs Used

- Google Maps API
- OpenWeatherMap API
- Stripe Checkout API

## 🔧 Tech Stack

- Frontend: React, Axios
- Backend: Node.js, Express
- APIs: Stripe, Google Maps, Weather API
- Deployment: Vercel + Render
